# 🧮 Calculator

A clean, minimal calculator built with HTML, CSS, and JavaScript — featuring a sleek dark UI with a glassmorphism-inspired design.

## 📸 Preview

> A dark-themed calculator with rounded buttons, glowing shadows, and an orange equals button.

## ✨ Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Percentage ( `%` ) support
- Delete last character ( `DEL` )
- Clear all ( `AC` )
- Decimal point input
- Responsive button layout

## 🛠️ Built With

- **HTML** — structure
- **CSS** — styling (glassmorphism dark theme, Google Fonts - Poppins)
- **JavaScript** — calculator logic

## 📁 Project Structure

```
calculator/
├── index.html    # Main HTML file
├── style.css     # Styles
└── script.js     # JavaScript logic
```

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/aajhazeen2-maker/calculator.git
   ```
2. Open `index.html` in your browser — no dependencies or build tools needed!

## 📌 Notes

- Uses JavaScript's `eval()` for expression evaluation.
- Designed for desktop browsers.
